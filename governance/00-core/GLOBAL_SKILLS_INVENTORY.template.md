# GLOBAL_SKILLS_INVENTORY

## Purpose

记录当前项目 / 用户环境中常用 skills 的用途分类与维护规则。

此文件不应携带旧项目事实，应只保留：

- 分类方式
- 推荐保留策略
- 维护规则

## Recommended Sections

### 1. Skills Layout

- skills 发现入口放哪里
- skills 真实目录放哪里
- 项目级与全局级如何区分

### 2. High-Frequency Skills

- 规划 / 编排
- 调试 / 测试
- 开发
- 文档 / Office
- 设计 / 视觉

### 3. Categorized Inventory

- 规划 / 编排 / 任务推进
- 审查 / 质量门禁
- 调试 / 测试 / 验证
- 通用工程开发
- 移动端 / 原生 / 图形专项
- 文档 / 表格 / 演示 / 文件处理
- 需求 / 产品 / 设计提示
- 知识发现 / 代码库理解 / 记忆
- Git / 发布 / 提交流程
- skill 开发 / skill 管理

### 4. Keep / Optional / Specialized

- 必保留
- 可选保留
- 特殊专项保留

### 5. Maintenance Rules

1. 新 skill 优先决定装到全局还是项目级
2. 已迁到全局的 skill 不在项目里保留副本
3. 大规模安装 / 迁移 / 清理后更新本文件
